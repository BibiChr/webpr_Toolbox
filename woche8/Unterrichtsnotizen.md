## Types in JS
- built-in types:
  - boolean
  - number
  - string
  - object
  - BigInt
  - Symbol
  - Function
  - null / undefined
- built-in types unterscheiden sich, in dem die Methode ```typeof``` nur diese zurückgeben kann
